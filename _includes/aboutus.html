About us

Javascript Dojo:Cheltenham is a group to socialise, learn, practice and build with javascript and beyond

Is anyone around Cheltenham or nearby areas interested in learning, practicing together, or teaching javascript and other coding languages to others? This is a group for anyone who:

  • is learning javascript (and frameworks), other coding languages, and web technologies.
  • is working on some personal project using javascript.
  • who would like to work together on our group projects.
  • would like to go through together with other fellow learners online exercises/programmes such as - Free code camp - Codecademy - Codefights - Codewars
  • is a beginner javascript coder, working through these exercises.
  • is an intermediate or experienced javascript coder who would like to help others.

Let's meet up and code along with other fellow learners and coders in Cheltenham or nearby areas!
